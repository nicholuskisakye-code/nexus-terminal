# Nexus Terminal\n\nModular terminal dashboard. Includes 12 themes (6 presets + 5 custom + default). See assets/ for preview images.\n\n## Themes included\n\ndefault, neon_tech, matrix_green, cyberpunk_purple_blue, minimal_clean, glitch_hacker, skull_terminal, dark_red_blood, blue_ai_circuits, retro_80s_neon_grid, omni_gold_runes, electric_yellow_hazard\n\n## Quick start\n\n1. Install dependencies:\n\n``\npython -m pip install -r requirements.txt\npython -m pip install windows-curses  # if on Windows\n``\n\n2. Run:\n\n```\npython app.py\n```\n\n## How to change theme\nEdit config.json and set "theme" to any of the theme folder names in skins/.
